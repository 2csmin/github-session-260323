# 📘 Today I Learned

### 1. 오늘 배운 내용
- 접근 제어자 public, default, private 개념을 배웠다.
접근 제어자에 따라 필드 접근 가능 범위가 달라진다는 것을 배웠다.
public 은 다른 패키지에서도 접근 가능하다.
default 는 같은 패키지 내에서만 접근 가능하다.
private 은 같은 클래스 내부에서만 접근 가능하다.

### 2. 핵심 정리 (내 언어로)
- 접근 제어자는 데이터를 보호하기 위한 장치이다.
접근 제어자를 사용하면 외부에서 데이터에 접근하는 것을 막을 수 있다.

### 4. 느낀 점
- 접근 제어자를 통해 데이터에 접근할 수 없도록 제한할 수 있다는 점이 인상 깊었다.
클래스를 무작정 public 으로 두지 말고 적절한 접근 제어자를 사용하도록 주의해야겠다.
